DataMind AI transforms any raw dataset into a complete machine 
learning report — automatically. Upload a CSV, select your target 
variable, and get data profiling, model comparison, AI insights, 
and a downloadable PDF report in minutes.

---

## 🚀 Features

### Data Processing
- Upload CSV, Excel, or JSON files
- Automated data profiling and summary statistics
- Missing value detection and handling
- Duplicate record detection

### Visualizations
- Correlation Heatmap
- Histograms & Distribution Plots
- Box Plots
- Scatter Plots
- Pie Charts

### AutoML Engine
- Automatically trains and compares 9 ML models
- Selects best model based on performance metrics
- Supports both Classification and Regression tasks

### AI-Powered Insights
- Google Gemini AI integration for natural language insights
- Chat with your dataset
- Automated PDF report generation

## ⚙️ Installation & Setup

```bash
# 1. Clone the repository
git clone https://github.com/aditya-0801/DataMind-AI.git
cd DataMind-AI

# 2. Install dependencies
pip install -r requirements.txt

# 3. Add your Gemini API key
# Open app.py and replace:
# GEMINI_API_KEY = "your_api_key_here"

# 4. Run the application
python app.py

# 5. Open in browser
http://localhost:5000
```
